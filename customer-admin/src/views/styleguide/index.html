드롭다운

기본 (.c-dropdown)

테두리가 있는 표준 드롭다운. 검색 필터/상태별 조회 등 폼 안에서 사용.

Ghost (.c-dropdown--ghost)

테두리 없는 드롭다운. 리스트 패널 헤더의 정렬/페이지 사이즈 등.

우측 정렬 패널 (style="right:0; left:auto;")

드롭다운이 컨테이너 우측 가장자리에 있을 때 사용. 패널이 트리거 우측에 맞춰 펼쳐짐.

Full border (.c-dropdown--full-border)

기본 패널은 트리거와 이어지지만, 이 모디파이어는 패널 상단도 보더가 보이는 독립 카드 형태.

콘텐츠 박스 (FAQ Q/A 상세)

컨테이너 (.content-bar)

세로 흐름 콘텐츠용 화이트 박스. .dashboard-bar와 동일한 외형(8px radius, shadow, margin-bottom 16px)이지만 flex가 아닌 일반 블록 컨텍스트라 헤더/Q/A 같은 멀티 행 콘텐츠를 자연스럽게 쌓을 수 있음. 패딩 24px 30px.

작성자 메타 (.user-id--lg)

기본 .user-id(13px) 대비 닉네임을 17px / weight 600으로 강조하는 모디파이어. 24px 아바타 + 닉네임 + 작성일 패턴으로 게시글 상세 헤더에 사용.

Q (.qa-item--center)

Q 배지 + 질문 텍스트(.qa-item__text--q, 17px / 600 / primary)를 세로 중앙 정렬, 간격 15px로 묶는 모디파이어.

A (.qa-item__answer + .qa-item__answer-text)

#F9F9F9 배경, padding 30px, 8px radius의 회색 박스. A 배지가 박스 안쪽 좌상단에 위치하고 텍스트는 .qa-item__answer-text로 감싸 우측 정렬.
